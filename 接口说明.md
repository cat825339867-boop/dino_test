如果label_name是sink
那么
label_store_dir = ./faiss_label_store/sink
index_path      = ./faiss_label_store/sink/val.faiss
mapping_path    = ./faiss_label_store/sink/id_to_path.json

/faiss/search 会默认去找：

faiss_label_store/
  └── sink/
      ├── val.faiss
      └── id_to_path.json

val.faiss由build_val_faiss_index.py 中的 save_outputs 生成，结构如下

output_dir/
├── val.faiss          # 向量索引
├── id_to_path.json    # id → 图片路径
└── features.npy       # 所有向量
